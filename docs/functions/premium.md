# 🔑 正版验证

:::caution
正版验证会丢失您的游戏数据，请联系管理员迁移数据。  
非正版请勿操作！
:::

## 操作方法

执行此指令即可：

```plain
/premium
```

## 正版玩家权益

1. `[正版]` 称号 **(如果正版验证后没有此称号，请输入 `/tags` 手动启用，如果 `/tags` 中没有，请联系管理员)**
2. 20 个领地玩家 **(由10个变为20个，您可以使用 `/lands edit <领地名>` 和 `/lands trust <玩家名>` 邀请其他玩家)**
3. 修改玩家名保留游戏数据 **(请在 Xbox个人中心 修改玩家名)**
