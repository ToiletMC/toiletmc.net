## 服务器地址

你可以使用 1.9-1.19.3 版本的客户端进入游戏，但为了你能够更好地体验游戏最新内容，我们推荐你使用最新版本进入游戏

服务器版本：[PaperSpigot](https://papermc.io) 1.19.3 不强制使用正版，但是你可以 [正版验证](functions/premium)

**Java版**服务器地址：`play.toiletmc.net` 或 `toiletmc.net`
