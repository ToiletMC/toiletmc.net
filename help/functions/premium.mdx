# 🔑 正版验证

:::caution
非正版请勿操作，会导致无法进入游戏！  
正版验证会丢失您的离线模式游戏数据！如需迁移游戏数据，请联系管理员。  
:::

## 操作方法

执行此指令即可：

```plain
/premium
```

## 正版玩家权益

1. 正版称号（自动发放，称号会显示在名字前面）
2. 皮肤显示（无需任何 Mod 即可显示正版皮肤和披风）
3. 免密登陆（加入服务器不再需要密码登陆）
4. 更多领地成员（由10个变为20个，您可以使用 `/lands edit <领地名>` 和 `/lands trust <玩家名>` 邀请其他玩家）
5. 修改玩家名保留游戏数据（请在 [minecraft.net](https://www.minecraft.net/en-us/msaprofile/mygames/editprofile) 修改玩家名后重新正版认证）
